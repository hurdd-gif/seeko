SEEKO Studio — Dashboard Guide

This guide walks you through the SEEKO Studio dashboard. Each section matches what you see in the optional in-app tour, so you can use this as a reference anytime.


Overview

Overview is your home screen. You see:

- Stat cards — Open tasks, completed count, team size, and number of documents. Open Tasks is clickable and takes you to the Tasks page.
- Game Areas — If your workspace uses areas, each area shows name, phase, description, and progress. Use this to see how different parts of the project are moving.
- Upcoming Tasks — Your next few open tasks (or "All caught up" if you have none). Click through to open task details.
- Recent Activity — Latest actions from the team (task updates, new docs, etc.) so you stay in the loop.


Tasks

Tasks is where you view and manage your work.

- My Tasks — You see your assigned tasks. Use search and filters (e.g. by status) to find what you need. Open any task to see full details, assignee, due date, linked document, and status.
- All Tasks (admins only) — Same list but for the whole team. You can manage and assign tasks to others.

Open a task to update status, change assignee, link a doc, or add notes.


Team

Team shows who's in your workspace and how to get in touch.

- Members — List of team members with name, department, and contact. Use this to see who does what and how to reach them.
- Contractors — If your workspace uses contractors, they appear in a separate section.
- Inviting people (admins only) — Use the invite form on the Team page to send an invite code by email. New members sign in with that code, set a password, complete onboarding, then see the dashboard.


Docs

Docs are shared documents and resources for the team.

- Browse and open team documents, specs, and other shared files.
- Create, edit, and organize docs as allowed by your role. Use them to keep specs, notes, and references in one place.


Activity

Activity shows recent actions and updates from your team.

- See who did what and when (task changes, new docs, team changes, etc.).
- Use it to catch up or dig into a specific change. More detail is available on individual activity items where applicable.


Settings and notifications

- Settings — In the sidebar (desktop) or via your profile (mobile): update your profile, display name, and sign out.
- Notifications — The bell icon in the sidebar (or header on mobile) shows recent notifications. Check it for updates that matter to you.


You can replay the in-app tour from the dashboard if you want a quick visual walkthrough again.
